Este trabajo esta realizado mediante el programa "Unity" por lo que el fichero que adjunto es un
paquete con todo el trabajo realizado, este paquete se abrirá con el Unity.

Para ello:

1- Creas un nuevo proyecto
2- En la pestaña "Assets" das a "Import package" -> "Custom Package"
3- Seleccionas el paquete y añades todo
4- Una vez importado entras a la carpeta "Scenes" y abres la que se llama "Juego"
5- Por último, para ejecutar el juego le das click al play en la zona superior 

Si le das a "Maximize On Play" antes de ejecutarlo se ejecuta agrandando la pantalla
(estando dentro de la pestaña "game")

Haré capturas de cada paso y lo guardaré en la carpeta "Capturas importar paquete"

Para empezar a jugar haces click en la pantalla y luego mediante las flechas del teclado
controlas el movimiento de la "pala" con la que golpeas la pelota.

Tienes 3 vidas, cada vez que la pelota cae al suelo pierdes una vida. 
Si llegas a 0 vidas pierdes y el juego se reinicia.
Si acabas con todos los bloques ganas y el juego se reinicia.

Dentro de la carpeta "Materials" tienes los materiales creados para la creación del juego.
Colores de bloques, particulas, paredes...
En la carpeta "Prefabs" se guardan conjuntos de creaciones que mediante los scripts se invocan.
Por ejemplo, el conjunto de bloques, la pala y la pelota...
En la carpeta "Scripts" están todos los scripts creados para el programa.
