El proyecto se abre mediante el fichero "index.html".
La interfaz consta de una zona para introducir datos y añadir funciones que se muestran
en el lado derecho, comenzando con la "Función 0". Luego tienes las opciones de "borrar"
y "modificar":

- borrar: introduces el número de la función a borrar y el programa te la elimina y la quita
	de la lista de funciones
- modificar: introduces el número de la función a modificar y el programa cambia la función 
	correspondiente

Por otra parte tenemos una zona de creación de fractales con los ejemplos de:

- Spierpinsky
- Mandelbrot
- Koch
- Tree Fractal

Además de esto tenemos las opciones "pinta", "limpia", "limpia pantalla", "background":

- pinta: dibuja en pantalla según los datos que hayas introducido
- limpia: borra las funciones y limpia todo lo que haya dibujado en el canvas
- limpia pantalla: simplemente limpia lo que haya dibujado poniendo de fondo el color que hayas
		asignado en el background
- background: eliges un color y al hacer click en background te cambia el fondo de ese color

Por último, hay una opción "color" que sirve para cambiar el color de lo que se pinta.

El entorno de programación usado ha sido Visual Studio Code y todo el programa
esta dentro del fichero "p3.js".
