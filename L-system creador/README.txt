El proyecto se abre mediante el fichero "index.html".
La interfaz consta de una zona para introducir los valores de los distintos parámetros:
Longitud, Ángulo, Axioma, Regla y Generación
y una zona en la que se muestran las producciones creadas.

Una vez abierto el "index.html" puedes cambiar todos los datos y se irá modificando tu L-System

El entorno de programación usado ha sido Visual Studio Code y todo el programa
esta dentro del fichero "p4.js".
