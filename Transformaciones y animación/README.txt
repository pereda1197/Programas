El proyecto se abre mediante el fichero "index.html" aqui tienes la opción "START" con la que se enciende
el canvas para poder dibujar.
Una vez pintas algo a la izquierda tienes las opciones "TRASLADAR", "ESCALAR" y "ROTAR" con sus variables
para introducir. Al pulsarlas se dibuja la figura que habías hecho pero con el movimiento que has elegido.
Por último, a la derecha tienes la opción "VER ANIMACION" que te ejecuta la animación creada.

El entorno de programación usado ha sido Visual Studio Code y todo el programa, incluida la animación
esta dentro del fichero "p2Final.js".

El formato de video de la animación es WMV Video File y para reproducirlo he usado el VLC.
